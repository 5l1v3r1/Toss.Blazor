# Toss.Blazor
Experimental project using AspNetCore Blazor

# Tech stack
- Blazor (0.4.0)
- Bootstrap 4
- Toastr
- Asp.net Core 2.1.1
- MediatR
- CosmosDB

# Feature list
- provide aspnet core identity pages (auth/open id, edit account) as Blazor page
- push new messages to the home page
- filter message by hashtag
